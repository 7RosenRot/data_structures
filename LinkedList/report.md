# Воссоздание списка по данным из файла `ListNode* recreate_list(const std::string &filename)`
Допустим, мы находимся  в самом начале воссоздания списка и считываем строку `apple;2`, получается, мы ссылаемся на пока еще не существующий элемент списка. В таком случае необходимо производить воссоздания за два прохода:
1. Читаем файл и выстраиваем связи между узлами. Заполняем поля `data`, `prev`, `next`, записываем полученные узлы в массив указателей `std::vector<ListNode*> nodes`, а указатели _(пока что в форме индексов элементов)_ в массив `std::vector<int> rand_index`.
2. На данный момент все узлы созданы, а значит можно перевести rand из формы индекса в указатель на соответствующий узел. Проходим по массиву узлов и записываем в поле rand указатель на нужный узел `nodes[i]->rand = nodes[rand_index[i]]`.

# Сериализация, запись данных из памяти в бинарный файл `void serialize(ListNode* head, const std::string &filename)`
Поскольку после завершения программы старые адреса в памяти могут быть перезаписаны, необходимо сохранить их в виде логических идентификаторов и присвоить каждому номер соответствующего узла. Для выстраивания _связи сырой адрес <-> номер узла_ используем структуру `std::unordered_map<ListNode*, size_t> list_nodes`. Алгоритм также работает в два прохода:
1. Проходим по всем элементам списка и записываем количество узлов.
2. Для каждого узла записываем _длину строки_, _содержание строки_, _индекс, указывающий на рандомный элемент списка (-1, если rand_index == nullptr)_

# Десериализация, восстановление списка из бинарного файла `ListNode* deserialize(const std::string &filename)`
По той же причине, что и в обычном алгоритме воссоздания списка, мы выполняем восстановления в два прохода:
1. Считаем общее количество записей, если число превышает `1000000` - останавливаем работу. Узнаём длину считываемой строки, если длина превосходит `1000` - останавливаем программу и освобождаем выделенное, для всех предыдущих элементов списка, место. Заполняем поля `data`, выполняем связывание элементов, заполняя поля `prev` и `next`.
2. Заполняем поле `rand` указаетлем на элемент, стоящим под соответствующим индексом.
